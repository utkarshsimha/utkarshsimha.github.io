Deep Learning for Speech Recognition in CMU Sphinx

Interned at Carnegie Mellon University under Prof. Bhiksha Raj and Prof. Rita Singh

Problem Statement : To use a Deep Neural Network in place of the existing Gaussian Mixture Model to predict Hidden Markov Model states in CMU Sphinx Speech Recognition System

My task : To implement a Deep Neural Network which uses the extracted features from the speech utterance and predicts the appropriate HMM state for the corresponding phoneme

Work done :

  • Implemented a Neural Network using the Theano library
  • Performed stochastic gradient descent using mini-batches, over the MNIST dataset. Acquired 97% testing accuracy
  • Upon trying the same model for the AN4 speech database, the accuracies were low
  • I performed hyper-parameter optimization using Optunity. Upon performing k-fold cross validation, I identified a set of parameter values for which the loss was the least. The accuracies increased, but was not sufficient.
  • Upon reading few papers on DNN-HMM models I implemented techniques such as momentum, dropout, learning rate decay, sparse auto-encoders and by changing the cost function to a Cross-Entropy function, the accuracies increased significantly
  • Further increase in accuracies were achieved upon using Yann LeCun's techniques like shuffling inputs, whitening inputs and also upon running it for longer epochs. Currently, the DNN model has a training accuracy of 81.08% and testing accuracy of 69.1%

Status : Currently implementing a technique to try and extract patterns in the phoneme boundaries to try and perform better training
