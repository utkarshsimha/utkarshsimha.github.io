Research Projects

  • Dimensionality Estimation (Master's thesis):

    • The project aims at estimating the intrinsic dimensionality of the low-dimensional manifold of a given dataset
    • For example, given a very high dimension (say 1024x1024) pictures of a teapot turning on a turntable, we want to find the axis of rotation (which is a 2-D circle).
    • For this, we use a statisical method based on the initialization used by KMeans++ and scale it for huge amounts of data on Apache Spark.
    • Code for this can be found here

  • Automated question generation based on lecture transcripts:

    • The project aims at generating objective type questions for video lectures
    • Explored various techniques that might work for this task, including Latent Dirichlet Allocation and Recurrent Neural Networks
    • Used part-of-speech and named-entity-recognition taggers to identify specific parts of the sentences, and used a knowledge graph (ConceptNet) to generate the questions
    • Code for this can be found here

  • Deep Neural Networks for speech recognition

    • The project aims at using deep neural networks for the task of automatic speech recognition
    • Tried various architectures such as vanilla feed-forward neural networks, end-to-end recurrent neural networks etc.
    • The final architecture was based on a end-to-end recurrent neural network which uses Connectionist Temporal Classification (CTC) loss and a beam search mechanism for predicting phonemes
    • Code can be found here


Personal Projects

  • Acoustic Embeddings for speech recognition

    Built a Variational Autoencoder to construct acoustic embeddings at a word level for the task of speech recognition. The embeddings tries to map acoustically similar words together. This can be used to improve the performance of the speech recognizer in noisy environments. Code for this can be found here.

  • Adversarial Expamples on Inception

    Constructed adversarial examples which try to fool the Inception model, based on this blog. The iPython notebook for this can be found here.
    Plan to improve this by making it robust to transformations, and also build a tutorial in PyTorch.

  • Northern Lights: A Simple Personal Assistant

    Built a simple personal assistant by stitching together multiple APIs available. This uses Snowboy from kitt.ai for hotword detection, Google Speech Recognition (online) and CMU Sphinx (offline) for speech recognition, Stanford NLP for performing some basic NER tagging and Wolfram API for knowledge querying Plan to improve it in the future by adding face and voice fingerprinting for personalization, as well as intent mining and NLU to provide more functionality.
